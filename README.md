# 💎 VelourEssence - E-commerce de Perfumería

Sistema web completo de comercio electrónico enfocado en la venta de perfumes, desarrollado bajo arquitectura por capas y patrón MVC, simulando un entorno real de producción.

---

## 🚀 Características principales

- 🔐 Autenticación de usuarios (registro, login y recuperación)
- 🛒 Carrito de compras dinámico
- 💳 Sistema de pagos (tarjeta y efectivo)
- 📦 Gestión completa de productos (CRUD)
- 🎯 Sistema de promociones automáticas
- ⭐ Sistema de reseñas de productos
- 🎨 Personalización de productos
- 📊 Panel administrativo / dashboard
- 🧾 Gestión de pedidos y compras
- 🌐 Internacionalización (multi-idioma)

---

## 🛠️ Tecnologías utilizadas

### Backend
- C#
- ASP.NET Core (MVC)
- Entity Framework

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Base de datos
- SQL Server

### Arquitectura y patrones
- Arquitectura en capas:
  - Application
  - Infrastructure
  - Web
- Patrón MVC
- Uso de DTOs
- AutoMapper
- Middleware personalizado
- Manejo de errores centralizado

---

## 📂 Estructura del proyecto

- `VelourEssence.Application` → Lógica de negocio y servicios
- `VelourEssence.Infrastructure` → Acceso a datos y repositorios
- `VelourEssence.Web` → Interfaz de usuario (MVC)
